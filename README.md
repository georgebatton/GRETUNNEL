# GRETUNNEL-CCNA 

Tools used **: Cisco Packet Tracer
Topics Covered**: GRE Tunnels, Static Routes, OSPF, Passive Interfaces  Basic CLI commands


## Topology
-4 PC's
-2 Switches
-4 Routers

## Configuration Steps
1. Created Subnets 10.0.1.0/24, 10.0.2.0/24, 100.0.0.0/30, 200.0.0.30 and 192.168.2.0/30, and GRE Tunnel 192.168.1.0
2. Connected tunnel across 192.168.2.0 via 100.0.0.0 and 200.0.0.0
3. Assigned ports to IP addresses and Subnets
4. Created static route on PTP connections
5. Tested connectivity within broadcast domains, OSPF AREA 0, and between different LANS.
6. Verified GRE header on outbound and inbound frames
